# Dashboard de Análisis Socioeconómico - ENAHO 2023

## Descripción del Proyecto
Este proyecto es una aplicación web interactiva/dinamica desarrollada con **Streamlit** y **Plotly**. Su objetivo es visualizar y analizar datos reales de la **Encuesta Nacional de Hogares (ENAHO) 2023** de Perú, enfocándose en la distribución demográfica y la relación entre el tiempo de trabajo y los ingresos económicos.

## Funcionalidades
- **Filtros Dinámicos**: Selección de datos por género a través de una barra lateral.
- **Histograma**: Visualización de la distribución de edades de los encuestados.
- **Gráfico de Dispersión**: Análisis de correlación entre horas laboradas e ingresos mensuales, incluyendo una línea de tendencia (regresión OLS).
- **Interactividad**: Gráficos dinámicos que permiten hacer zoom y ver detalles al pasar el mouse.

## Tecnologías Utilizadas
- **Python**: Lenguaje principal.
- **Pandas**: Manipulación y limpieza de datos.
- **Plotly Express**: Creación de gráficos interactivos.
- **Streamlit**: Desarrollo del dashboard y despliegue web.
- **Render**: Hosting de la aplicación en vivo.

## Instrucciones de Uso
1. Seleccione los filtros en la barra lateral izquierda.
2. Active las casillas de verificación para generar los gráficos deseados.
3. Explore los datos interactuando directamente con las visualizaciones.
